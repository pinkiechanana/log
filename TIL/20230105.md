Typescript 기본 syntax + React, TodoList 만들기

- `type` and `interface` : 

  - `type`: 
```
type A = {
  a: string;
  b: number;
}

type B = A & {
  c: string;
  d: number;
}

let y: B = {
  c: "lala",
  d: 34
}
// error : A & 이므로 a와b도 포함해야된다.
```

  - `interface` 
```
interface Person {
  name: string;
  age?: number;
}

interface Guy extends Person {
  profession: string;
}
// just like what you learned in Java
```

  - 둘이 같이 이용도 가능
```
interface Person extends A {
  name: string;
  age?: number;
}

type B = Person & = {
  c: string;
  d: number;
}
```
